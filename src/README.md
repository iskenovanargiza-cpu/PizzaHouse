# My Awesome Project

This is a console-based Java application for a custom pizza shop.  
Customers can create their own pizza orders, add drinks and garlic knots, and checkout with a receipt.

# Features

- Start a new order
- Add pizzas:
    - Choose size (8", 12", 16")
    - Choose crust type (thin, regular, thick, cauliflower)
    - Add toppings (meat, cheese, regular)
    - Choose sauces
    - Option for stuffed crust
- Add drinks (small, medium, large)
- Add garlic knots
- Checkout: shows order summary and total price
- Cancel order: deletes current order and goes back to main menu

# Pricing

- Pizza crust: $8.50–$16.50
- Meats: $1–$3 (+ extra meat $0.50–$1.50)
- Cheese: $0.75–$2.25 (+ extra $0.30–$0.90)
- Regular toppings & sauces: included
- Drinks: $2–$3
- Garlic knots: $1.50

# Receipts

- Orders are saved in `receipts/` folder
- Filename format: `yyyyMMdd-HHmmss.txt`
- Includes pizzas, toppings, drinks, garlic knots, and total cost
